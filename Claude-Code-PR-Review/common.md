# 공통 리뷰 기준 (모든 잡이 먼저 읽음)

## 대전제
- 이 레포는 SwiftUI + TCA(The Composable Architecture) 기반 iOS 앱이다.
- **이번 PR의 diff(새로 추가/수정된 부분)만** 리뷰한다. 기존 코드의 누적 문제는 지적하지 않는다.
- 코멘트는 한국어로, 문제가 있는 라인에 인라인으로 단다.
- 목적은 팀이 합의한 아키텍처·컨벤션을 깨는 변경을 조기에 잡는 것이다. 문서에 없는 새 규칙을 강요하지 않는다.

## 제외 (리뷰하지 말 것)
- lock/생성물: `Package.resolved`, `*.xcodeproj/project.pbxproj`의 단순 파일 추가 diff, `DerivedData/`
- 리소스: `Assets.xcassets/`, 이미지, 폰트
- CI가 이미 잡는 것: 포맷/린트(SwiftLint·SwiftFormat)/빌드 에러

## 심각도 & 분량
- ❗ **필수 수정**: 크래시·데이터 손실·보안 이슈·아키텍처 위반(TCA 원칙 파괴)
- 💊 **개선 제안**: 구조·중복·가독성 — 잡당 최대 5개, 초과는 "외 N건"으로 요약
- ❓ **질문·확인**: 의도가 불분명해 저자 확인이 필요한 것
- 차단급(❗) 이슈가 없으면 요약을 "차단할 이슈 없음"으로 시작
- 잘 작성된 부분이 있으면 짧게 칭찬해도 좋다
